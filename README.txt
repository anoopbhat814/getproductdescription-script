# purpose of project :
this project scrapes the description of the products from given url and writes the correct descrption between
specified limit of words with the help of chat gpt

you have to enter your own chat gpt key in the .env file for the script to be working properly.

#Running the Script

enter into the project folder
1. cd nodescript 

install the dependencies
2. npm install

run script 
3. node app.js

# we have added a example sheet which contains urls for some aliexpress products you can also add your own list
